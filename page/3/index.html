安卓相对布局属性小结

安卓的相对布局是一种非常灵活的布局方式，能灵活应对各种场景，本篇总结了相对布局 RelativeLayout中的标签的属性。

安卓相对布局属性小结

相对布局 RelativeLayout子标签可用属性

1.相对于父容器有关属性

  1. 于父窗体中央：centerInParent=true
  2. 于父窗体左侧：alignParentLeft=true
  3. 于父窗体右侧：alignParentRight=true
  4. 于父窗体上侧：alignParentTop=true
  5. 于父窗体下侧：alignParentBottom=true

2.相对于组件有关属性

相对于组件位置：

  1. 在组件上方：above=”组件id”
  2. 在组件下方：below=”组件id”
  3. 在组件左侧：toLeftOf=”组件id”
  4. 在组件右侧：toRightOf=”组件id”

与组件对齐方式：

  1. 上边缘对齐：alignTop=”组件id”
  2. 下边缘对齐：alignBottom=”组件id”
  3. 左边缘对齐：alignLeft=”组件id”
  4. 右边缘对齐：alignRight=”组件id”

3.自身对齐有关属性

  1. 水平方向对齐：center_horizontal=true
  2. 垂直方向对齐：center_vertical=true
